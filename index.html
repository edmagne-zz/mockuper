Crie e compartilhe protótipos para projetos de forma fácil e intuitiva.

O Mockuper provê uma interface amigável para geração de estruturas de projeto, além de permitir a escolha de padrões visuais dentre modelos pré-definidos. Seus diferenciais estão na ferramenta para compartilhamento e colaboração - que mantém histórico de revisões e comentários dos clientes - e na possibilidade de definição de uma paleta de cores customizada para criação de temas totalmente adaptados à sua necessidade.

            mockuper.people.getNames(function() {
              for (i in mockuper.people.names) {

Pegue pesssoas específicas da lista

          mockuper.people.getNames(function() {
            for (i in mockuper.people.names) {
              $('#specific-list').append($('
').html(mockuper.people.names[i]));
            }
          },{array: [1, 0, 4]});

Apenas um número de pessoas aleatórias

          mockuper.people.getNames(function() {
            for (i in mockuper.people.names) {
              $('#random-number-of-people-list').append($('
').html(mockuper.people.names[i]));
            }
          },{random: true,length: 3,});
